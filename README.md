# Jhow Store Server
This is the server for a fake store called Jhow store created using [Node.js](https://nodejs.org/) and Express [Express](https://expressjs.com). The webapp code for the app is located at [jhow-store-webapp](https://github.com/jonathan-santos/jhow-store-webapp)


## Running the app
First, install the NPM packages with:

```bash
npm i
# or
yarn
```

And then run the development server with:

```bash
npm run dev
# or
yarn dev
```
